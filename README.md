# Reto

Este proyecto es una aplicación web para la gestión de reservas de salas.

## Estructura
- index.html
- script.js
- styles.css

## Instrucciones para subir a GitHub

1. Inicializa el repositorio:
   ```bash
   git init
   ```
2. Añade los archivos:
   ```bash
   git add .
   ```
3. Haz el primer commit:
   ```bash
   git commit -m "Primer commit"
   ```
4. Crea un repositorio en GitHub y sigue las instrucciones para conectar el repositorio remoto:
   ```bash
   git remote add origin https://github.com/usuario/nombre-repo.git
   git branch -M main
   git push -u origin main
   ```

## Notas
- Asegúrate de reemplazar la URL del repositorio por la tuya.
- Puedes agregar un archivo .gitignore si es necesario.
